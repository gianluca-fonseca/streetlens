Steps verified (screenshot)
---------------------------
1. Workbench renders an explicit "Enlarge frame N" magnifier on EVERY thumbnail
   (both segment filmstrips and the unmatched section). Click-to-inspect on the
   thumbnail body is unchanged.                                 [u1-00]

2. OPEN: the magnifier on frame 0 opens the full-size lightbox — dialog
   role=dialog aria-modal, counter "1 of 7", caption "Frame 0 / esc-sa-0001",
   the model rationale, prev/next controls.                     [u1-01]

3. NAVIGATE: ArrowRight -> "Frame 1 / 2 of 7" with its own rationale. Keyboard
   arrows drive the same nav as the on-screen chevrons.         [u1-02]

4. WRAP: from Frame 0, ArrowLeft wraps to the last frame in seq order — asserted
   programmatically: data-frame-seq="6", counter "7 of 7". Nothing skipped (all
   7 frames, unmatched frame 5 included, are navigable).

5. CLOSE + FOCUS RETURN: Esc closes the overlay and returns focus to the
   "Enlarge frame 0" button that opened it — asserted: document.activeElement
   aria-label === "Enlarge frame 0". (X button and backdrop tap also close.)

6. INSPECTOR IMAGE: clicking the inspector's own image (the surface a map-dot
   selection routes to) opens the lightbox on that frame — asserted: the image
   is a button[data-expand-seq] whose click opens data-lightbox frame 2. The
   inspector shows the enlarge affordance on hover.             [u1-03]

7. TOMBSTONE: delete frame 2 via the inspector (Delete -> Delete it; POST
   /api/admin/capture/frame 200, local tombstone), then enlarge it. The lightbox
   shows the tombstone PLACEHOLDER, not a broken image — asserted: no <img>
   inside the dialog, placeholder text "This frame was deleted for privacy...",
   and a "Deleted" state badge in the caption.                  [u1-04]

8. MOBILE: at 390x844 the overlay fills the viewport, the image fits to screen,
   and the caption/nav/close stay reachable.                    [u1-05]

9. ES LOCALE: /es/... renders every new string in Spanish — asserted:
   aria-label "Cuadro 0, a tamaño completo", counter "1 de 7", controls
   "Cerrar la vista a tamaño completo" / "Cuadro anterior" / "Cuadro siguiente".
